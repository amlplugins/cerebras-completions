# @amlplugins/cerebras-completions

Native `@cerebras/cerebras_cloud_sdk` SDK re-exported under the `@amlplugins` namespace.

> Cerebras Text Completions — legacy completion endpoint for raw prompt continuations on Wafer-Scale Engines.

## Install

```bash
npm install @amlplugins/cerebras-completions
```

## Use

```ts
import { /* named exports from @cerebras/cerebras_cloud_sdk */ } from "@amlplugins/cerebras-completions";
// or
import sdk from "@amlplugins/cerebras-completions";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
